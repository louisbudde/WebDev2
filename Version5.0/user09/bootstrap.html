About Me

My name is Louis Budde and I attend Shakopee High School. In this project we used the bootstrap framework to make our own personal website. We made grids, brought in images, used specific classes and psuedo-classes, and much more. This project unlocked something in my brain, making this whole coding thing a lot easier.

About Bootstrap

Bootstrap is a open-source CSS framework. It was created Mark Thornton and Mark Otto, who were working with twitter at the time. It was released as an open source project on August 29, 2011. It is still going strong, in fact Bootstrap Version 5 was released in May, 2021.
